An Introduction to JavaScript - Part 2

Types of values

Before we delve into the world of variables/arrays/objects/etc, we will explain some of the types of values:

  • Strings are groups of letters strung together. However, strings can consist of only one letter. Example: "This is a string"
  • Integers are numbers without any decimal points. Example: 42
  • On the other hand, floats are numbers with a decimal point. Example: 12.012

Variables

What are variables?

I'll use an analogy for this question. Let's say that variables are storage boxes:

|Box|

The "boxes" (variables) store values instead. Here's another diagram to illustrate that:

"Hello world" ---> |Box|

Oh. But how do I make and modify variables?

It's quite simple! Open up the console (this should be explained in the previous post) and type this in:

var x = 10;

To explain what the equal sign does, I'll raise an example of using the "=" sign in JavaScript: x = y in JavaScript will set the value of x to y.

Afterwards, type this in and see what happens:

alert(x);

Why does that happen?

Let's use the box analogy again (diagrams in bold):

var x = 10;
10 ---> |Box|


alert(x);
|Box| ---> 10 ---> alert() ---> Click here for result

I see. Are there ways to put more than one value in a variable?

Yes. You can store multiple values in something called an array. I don't think I need to explain how it works.

Cool, but how do you use arrays?

To make a new array:

var myarray = ["item1", "item2"]

How do you access particular elements to your array?

This is how you get an element in your array:

myarray[elementnumber]

It's not working! I put in myarray[1] and the 2nd item in my array came up instead!

The 1st item of the array is myarray[0], the second item of the array is myarray[1], and so on.

How do you make new items in the array anyway?

This is a good time to introduce the golden rule of arrays and variables:

If the value(s) in an array/variable don't/doesn't exist, the previously non-existent array/variable/array item will be automatically created before proceeding.

In this command, the 20th element in the array will be created (if it was non-existent) before being assigned a value:

myarray[20] = "somevalue";

In fact, when declaring arrays/variables, you can leave out the var completely.

Summary

  • Strings are letter(s), declared with quotes on either end of the string
  • Integers are whole numbers
  • Floats are numbers with decimal places
  • To declare a variable: variable_name = initial_value; Example: myvariable = 10;
  • To declare an array: array_name = [value1, value2, value3]; Example: mypicnic = ["fruit", "vegetables", "bread"];
