My blog layout

Hello again! I'm going to show you how to use my blog layout I am currently using right now!

  1. Firstly, put this in your head: <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" type="text/css" href="../blog.css">
    <link href='http://fonts.googleapis.com/css?family=Open+Sans' rel='stylesheet' type='text/css'>
  2. Put your content in the div tag with the "content" ID.
  3. If you want navigation, put this code after the "content" div: <a href="#linktolastpost" id="previous">Previous post</a>
    <a href="#linktonextpost" id="next">Next post</a>


    If you're applying the navigation links to the first post, remove the first <a> tag. If you're applying the navigation links to the latest post, remove the second <a> tag.

Wait, that's it?

Yep! Just use <code> for codeblocks and you'll be fine.
