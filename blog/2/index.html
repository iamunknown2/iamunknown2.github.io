An Introduction to JavaScript - Part 2

Variables

What are variables?

I'll use an analogy for this question. Let's say that variables are storage boxes:

|Box|

The "boxes" (variables) store values instead. Here's another diagram to illustrate that:

"Hello world" ---> |Box|

Oh. But how do I make and modify variables?

It's quite simple! Open up the console (this should be explained in the previous post) and type this in:

var x = 10;

Afterwards, type this in and see what happens:

alert(x);

Why does that happen?

Let's use the box analogy again (diagrams in bold):

var x = 10;
10 ---> |Box|


alert(x);
|Box| ---> 10 ---> alert() ---> Click here for result

I see. Are there ways to put more than one value in a variable?

Yes. You can store multiple values in something called an array. I don't think I need to explain how it works.

Cool, but how do you use arrays?

To make a new array:

var myarray = ["item1", "item2"]

How do you access particular elements to your array?

This is how you get an element in your array:

myarray[elementnumber]

It's not working! I put in myarray[1] and the 2nd item in my array came up instead!

The 1st item of the array is myarray[0], the second item of the array is myarray[1], and so on.
